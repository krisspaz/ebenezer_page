# Ebenezer Cobán Mobile App

App nativa móvil (Android & iOS) construida con **Kotlin Multiplatform**.

## Estructura del Proyecto

- `shared/`: Código compartido (Lógica, Modelos, Traducciones, Tema)
- `androidApp/`: Aplicación Android nativa (Jetpack Compose)
- `iosApp/`: Aplicación iOS nativa (SwiftUI)

## Cómo empezar

1. Abre esta carpeta `mobile` en **Android Studio**.
2. Deja que Gradle sincronice el proyecto.
3. Ejecuta la configuración `androidApp` para ver la app en el emulador.

## Funcionalidades Implementadas (Shared)
- Multi-idioma (Español, Inglés, Q'eqchi')
- Tema Claro/Oscuro
- Persistencia de preferencias
